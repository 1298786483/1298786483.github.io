前言：关于白嫖的艺术与科学在这个流量比金子还贵的年代，每一个 NEUer 的月底账单里都藏着一部血泪史。面对 IPv4 计费、IPv6 免费这种“一国两制”的策略，一个伟大的念头不可避免地浮现：能不能把所有流量都伪装成 IPv6，实现合法的“零元购”？答案是肯定的。这不仅仅是一次省钱的尝试，更是一场关于Hysteria 2 协议、QUIC 拥塞控制与反 DPI 策略的硬核技术探险。准备好了吗？系好安全带，我们要飙车了。

搞机历程

第一章 环境侦察：为什么我们可以白嫖？1.1 CERNET 的馈赠我们要感谢中国教育网（CERNET）。为了推广 IPv6，教育网制定了IPv6 流量免费的政策。对于东北大学网络中心来说，计费系统通常部署在 IPv4 网关上，而 IPv6 数据包往往直接绕过了计费模块。这就是我们的阿基米德支点。1.2 为什么旧方法（OpenVPN/SSR）不行了？

传统 TCP 协议的软肋：特征明显：DPI（深度包检测）能轻易识别长时间的 TCP 连接。拥塞退让：校园网晚高峰丢包时，TCP 会礼貌地降低速度。在抢带宽的战争中，礼貌意味着卡顿。

1.3 新时代的救世主：Hysteria 2Hysteria 2 基于 UDP，修改了底层的 QUIC 协议。它的拥塞控制算法叫 Brutal（残忍模式）。不管丢包：即使网络丢包，我也要发数据，发到你收到为止。伪装：把自己伪装成标准的 HTTP/3 流量（看起来像是在访问 Google）。0-RTT：利用 UDP 无连接特性，秒开网页。第二章 军火采购：服务器（VPS）的选择要实现转发，你需要一台拥有 IPv4 + IPv6 双栈 的境外服务器。
    第三章 特别篇：DigitalOcean 注册避坑指南（本章高能，请仔细阅读，否则你的 $200 美刀会瞬间蒸发）很多同学拿到 GitHub 学生包后，兴冲冲去注册 DigitalOcean (DO)，结果还没创建服务器号就没了。这是因为 DO 的风控算法把你看作了“羊毛党”。3.1 准备工作GitHub Student Developer Pack：确保你已经通过了认证。点击申请纯净的 IP 环境：danger,禁止 使用 VPN/代理注册。success,推荐 使用校园网 IPv4 或手机热点注册。IP 地址必须和你的物理位置（沈阳）一致。支付方式：你需要一张 Visa/MasterCard 信用卡（国内银行的双币卡即可）。或者 PayPal（需要先充值 $5 美元进去作为保护费）。3.2 注册流程 (Timeline)

    注册生死线

    3.3 创建 IPv6 Droplet

    重点中的重点：在创建 Droplet 页面，”Select additional options” 区域，务必勾选 [IPv6]。如果不选这个，你买的就是个纯 IPv4 服务器，校园网免流直接失败。

    第四章 服务端部署：一键脚本现在你已经是尊贵的 Linux 管理员了。用 SSH 工具（推荐 FinalShell 或 Termius）连接你的服务器。4.1 安装 Hysteria 2复制以下命令，回车（感谢开源社区大佬 flame1ce）：Bashwget -N –no-check-certificate https://raw.githubusercontent.com/flame1ce/hysteria2-install/main/hysteria2-install-main/hy2/hysteria.sh && bash hysteria.sh
    4.2 配置向导按屏幕提示操作：选择操作：输入 1 (Install)。证书类型：选 1 (Self-signed certificate 自签证书)。不要搞什么 ACME 域名证书，麻烦且容易出错。自签证书配合客户端指纹验证最稳。端口：输入 11451（或者其他高位端口）。密码：设置一个复杂的密码。4.3 获取配置信息安装完成后，终端会输出一堆红红绿绿的字。请把它们复制到记事本里保存好！Server IP: 你的服务器 IP（注意：你需要用 IPv6 地址，通常以 2400: 或 2001: 开头）。Port: 11451Password: ******Fingerprint: SHA256 指纹（用于客户端验证）。第五章 客户端配置：Clash Verge Rev (让魔法生效)告别丑陋的 v2rayN 吧。Clash Verge Rev 是目前 Windows/Mac 上最好用的客户端，支持最新的 Meta (Mihomo) 内核，完美兼容 Hysteria 2。下载 Clash Verge Rev5.1 基础设置安装并打开 Clash Verge Rev。点击左侧 设置 -> Clash 内核，确保显示为 Mihomo (Meta)。如果不是，点击右侧按钮下载/重启内核。开启 IPv6：在 设置 中找到 IPv6 选项，必须勾选！5.2 编写配置文件 (YAML)由于 Hysteria 2 协议较新，很多订阅转换不支持，我们手动搓一个配置。点击左侧 配置 -> 新建 -> 本地文件。文件名填 NEU_Hysteria2，保存。右键点击刚创建的文件 -> 编辑文件。全选并删除里面的默认内容，粘贴以下代码（记得修改成你自己的信息）：YAML# Clash Verge Rev (Mihomo) Hysteria 2 专属配置
    port: 7890
    socks-port: 7891
    mixed-port: 7890
    allow-lan: true
    mode: rule
    log-level: info
    ipv6: true # 必须开启

    dns:
    enable: true
    listen: 0.0.0.0:1053
    ipv6: true
    enhanced-mode: fake-ip # 推荐 Fake-IP 模式，省去本地 DNS 解析流量
    fake-ip-range: 198.18.0.1/16
    nameserver:
    - https://dns.alidns.com/dns-query
    - https://doh.pub/dns-query

    proxies:

    • name: “NEU_IPv6_Hy2”
      type: hysteria2

      务必填入你的 IPv6 地址，并且加上方括号

      server: “[2400:6180:0:d0::xxxx:xxxx]”
      port: 11451
      password: “你的密码”

      端口跳跃：对抗校园网 QoS 限速的神器

      填写一个范围，让流量在这些端口间横跳，防止被网管针对

      ports: 20000-50000

      混淆密码 (Obfs) - 如果服务端配置了 Salamander

      obfs: salamander

      obfs-password: “你的混淆密码”

      伪装域名

      sni: www.bing.com
      skip-cert-verify: true

      填入服务端生成的 SHA256 指纹，防止中间人攻击

      fingerprint: “xx:xx:xx:xx……”

      暴力拥塞控制 (Brutal)

      根据你的实际网速填写，比如你寝室是百兆口，就填 100 Mbps

      填太大会导致严重丢包！

      up: 50 Mbps
      down: 500 Mbps

    proxy-groups:

    • name: “节点选择”
      type: select
      proxies:
      • “NEU_IPv6_Hy2”
      • DIRECT

    rules:

    校内网直连 (防止访问教务处绕地球一圈)

    • DOMAIN-SUFFIX,neu.edu.cn,DIRECT
    • DOMAIN-SUFFIX,neu.edu,DIRECT
    • IP-CIDR,202.118.0.0/16,DIRECT
    • GEOIP,CN,DIRECT
    • MATCH,节点选择
      5.3 启动与验证保存文件，右键选择 启用。点击左侧 代理，选择 节点选择 -> NEU_IPv6_Hy2。点击 系统代理 开关。打开浏览器访问 ipv6.google.com。

      判定标准：如果能流畅打开 4K 视频，同时登录校园网认证后台，发现已用流量没有随视频播放而疯狂增加（只有微量的心跳包流量），恭喜你，越狱成功！

      第六章 常见问题 (FAQ)
      我配置完了，但是完全连不上？

      检查时间：Hysteria 2 对时间非常敏感，请确保电脑和服务器的时间误差在 15 秒以内。检查防火墙：服务器提供商（如 RackNerd）的后台控制面板里是否放行了 UDP 端口？服务器内部的 ufw 是否放行了？检查 IPv6：你的寝室网口真的有 IPv6 吗？在 CMD 输入 ipconfig 看看有没有 IPv6 地址。

      速度很快，但是玩游戏跳 ping？

      Hysteria 2 是基于 UDP 的，虽然暴力，但对游戏这种对抖动敏感的应用不太友好。建议游戏时关闭系统代理，使用专业的游戏加速器（通常走 Netch 模式），或者在规则里把游戏域名直连。

      结语配置完这一切，看着 Clash 面板上跳动的速率，你可能会有一种战胜了体制的快感。但这不仅仅是为了省那几十块钱网费。在这个过程中，你学会了 Linux 基础、理解了 TCP/UDP 的区别、搞懂了证书体系。这些知识，比你在《计算机网络》课上睡一觉学到的要多得多。Keep calm and carry on, but don’t get caught.(最后更新于 2025年11月，若学校修补漏洞，本教程原地自爆)
